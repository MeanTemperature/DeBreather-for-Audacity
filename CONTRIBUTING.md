### Contributing

This project is entirely open-source, and contributions are absolutely welcome! Whether you are a Nyquist wizard or an audio editor who found the perfect preset, feel free to jump in.

- **Found a bug?** Open an Issue with a screenshot of the error or a description of the audio behavior.

- **Have a code improvement?** Fork the repo, make your tweaks, and submit a Pull Request.

- **Found the perfect settings?** Start a thread in the Discussions tab if you've dialed in the exact right parameters for a specific microphone (like an RE20, an NT1, etc.) so others can use them!
